## 共建！

- 1. 你可以随时在此 GitHub 仓库进行 Commit ，在经过可用性、安全性以及合法性检查后都会同意合并的喵

- 2. 你可以通过加入 [QQ群](http://qm.qq.com/cgi-bin/qm/qr?_wv=1027&k=SgIg_V-48eLjTeaPIy6R9Q-6AnkYbLCw&authKey=67iI1I3PKwihloUweW2kQbnoU%2B3m2wGXBEBTwXOMYIJZ3W%2BDNLdCOMMeKYv7%2FYev&noverify=0&group_code=1074482483) 来发发资源或者直接在群里发链接，我会在空余时间更新网站喵

- 3. PR 注意事项

> 如果你需要在sites增加分类请告诉我并由我来增加分类喵
> 如果添加网站通过检查后会直接合并喵

## 项目目录说明

```text
url-tools/
├── .vitepress/             # VitePress 核心配置目录 （会拒绝合并请求）
│   ├── theme/
│   │   ├── index.ts
│   │   └── style.css
│   └── config.mts
├── public/                 # 静态资源目录
│   └── images/
│       └── logo.png        # 网站 Logo
│                             # md文档若需图片请上传到这里
├── sites/                  # 网站内容目录 (所有网站列表的 Markdown 文件)
│   ├── index.md            # 网站列表总览页
│   ├── resource.md         # 素材与资源分类
│   ├── music.md            # 音乐学习分类
│   ├── entertainment.md    # 休闲娱乐分类
│   ├── tools.md            # 在线工具分类
│   └── personal.md         # 个人与推荐分类
├── about.md                # 关于本站页面
├── announcements.md        # 公告发布页面
├── contributors.md         # 贡献者名单页面
├── help-us.md              # 贡献指南页面
├── index.md                # 网站主页
├── LICENSE                 # MIT 开源许可文件
├── package.json            # 项目依赖和脚本定义文件 (如果存在的话，npm install 依赖此文件)
└── README.md               # 项目说明文件 (你现在正在看的文件)
```
