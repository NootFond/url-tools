# 🎮 休闲娱乐喵

### 心理测试

- **[iyoushi (有事)](https://iyoushi.cn/survey )** - 提供各种有趣的心理测试，图一乐的好去处喵。
- **[Arealme](https://www.arealme.com/cn )** - 同样是内容丰富的在线测试网站，包含性格、智商等多种测试喵。

### 游戏相关

- **[A Dark Room](https://adarkroom.doublespeakgames.com/?lang=zh_cn)** - 小巧轻量的纯文字挂机小游戏（甚至还有剧情）。
- **[Enazo](https://enazo.cn/)** - 在线的你画我猜小游戏。
- **[ZType](https://zty.pe/ )** - 一款将打字与射击游戏结合的网页游戏，练习打字非常有趣喵。
- **[Aternos](https://aternos.org/ )** - 提供免费的《我的世界》服务器，支持 Java 版和基岩版喵。
- **[Chunk Base](https://www.chunkbase.com/ )** - 《我的世界》种子地图查看器，可以帮你快速找到各种遗迹和生物群系喵。
- **[MCAPKS](https://mcapks.net/ )** - 提供《我的世界》各种版本的下载喵。
- **[Littleskin](https://littleskin.cn/)** - 我的世界皮肤站。
