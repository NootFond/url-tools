# 🛠️ 在线工具喵

### 综合工具箱

- **[Cratex](https://cratex.app/zh-CN )** - 一个集合了各种好玩、实用小工具的网站喵。
- **[Caliper 工具集](https://caliper.pages.dev/ )** - 纯前端实现的 HTML 工具集，轻量、快速喵。
- **[一个好用的工具箱](https://www.cyhaoka.vip/ )** - 另一个多功能在线工具箱网站喵。
- **[前端哥工具箱](https://tool.qianduange.cn/)** - 前端哥工具箱，但是不止前端。

### 媒体处理

- **[iLoveIMG](https://www.iloveimg.com/zh-cn/ )** - 强大的在线图片处理工具，可以提高图片分辨率、压缩、转换格式这些喵。
- **[路过图床](https://imgse.com/ )** - 一个免费、稳定的图片托管服务（图床）喵。
- **[SnapAny](https://snapany.com/ )** - 支持从多个主流平台提取和下载在线视频喵。
- **[AI-MIDI](https://ai-midi.com/ )** - 利用 AI 技术将音频文件（如 MP3）转换为 MIDI 文件喵。
- **[开源 TTS 文本朗读](https://huggingface.co/spaces/k2-fsa/text-to-speech )** - 在线的文本转语音（TTS）工具，可以生成朗读音频喵。

### 文本与创意

- **[反向词典 (WantWords)](http://wantwords.net/ )** - 当你想不起来一个词怎么说时，可以输入它的意思来查找词语喵。
- **[花样字体转换器](https://charactercalculator.com/zh-cn/font-changer )** - 可以将普通文字转换成各种特殊符号组成的花样字体喵。
- **[取名网站 (NameChef)](https://www.namechef.co/cn/ )** - 辅助你取中文名或英文名的在线工具喵（何意味）。

### 游戏开发
- **[Twine2](https://twinery.org/)** - 一款可视化流程图的文字游戏编辑器，除了应用版和网页版之外还拥有CLI工具。
- **[Phaser](https://phaser.io/)** - 一款基于web前端的游戏引擎，很强大（但你不能拿它和unity和unreal这种比呀）。

### 其他

- **[GitHub Proxy](https://github.akams.cn/)** - GitHub文件加速工具，快！
- **[Cpolar](https://dashboard.cpolar.com/)** - 免费的内网传透工具，大善人。
- **[TheCatAPI](https://thecatapi.com/ )** - 一个可以免费获取小猫图片的 API 接口，适合开发者喵。
- **[QQ空间好感度查询](https://h5.qzone.qq.com/close/rank )** - 查询你与 QQ 好友的亲密度排行喵（wow太酷了喵）。
